## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run startw

# watch mode
$ pnpm run startbw

# production mode
$ npm run start:all
```

## update version

`변경사항 모두 커밋 완료 필요`

npm version patch  # 기본 버전 증가
npm version minor  # 부 버전 증가
npm version major  # 주 버전 증가
